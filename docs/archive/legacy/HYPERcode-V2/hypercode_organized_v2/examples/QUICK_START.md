# 🚀 Quick Start - Try HyperCode in 60 Seconds!

## 3 Simple Demos (Ready to Run!)

### Demo 1: Hello World
**File**: `demo_hello.hc`  
**What it does**: Prints friendly messages  
**Run it**:
```bash
python -m src.hypercode examples/demo_hello.hc
```

### Demo 2: Fibonacci
**File**: `demo_fibonacci.hc`  
**What it does**: Calculates Fibonacci numbers using recursion  
**Run it**:
```bash
python -m src.hypercode examples/demo_fibonacci.hc
```

### Demo 3: Todo List
**File**: `demo_todo.hc`  
**What it does**: Manages tasks with variables and conditionals  
**Run it**:
```bash
python -m src.hypercode examples/demo_todo.hc
```
